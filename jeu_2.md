# Tutoriel 2

## @showdialog

Programme un lutin pour qu'il se déplace sur l'écran.

## Étape 1

Ajoute le bloc ``||scene:définir la couleur d'arrière-plan||`` (onglet ``||scene:Scène||``) dans le bloc ``||loops:au démarrage||``.

Choisis une couleur.

```blocks

scene.setBackgroundColor(1)

```
